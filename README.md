# SEO Guest Posting AI Agent

An open-source AI agent that automates SEO and guest posting workflows.

## 🎯 Problem

SEO and guest posting workflows are highly manual and time-consuming. 
Marketers spend hours on keyword research, content planning, and 
outreach emails.

## 💡 Solution

This AI agent automates:
- **Keyword Research** — generates relevant SEO keywords for any niche
- **Content Briefs** — creates structured content outlines automatically
- **Outreach Emails** — generates personalized guest posting outreach emails

## 🛠️ Tech Stack

- Python
- LangChain (planned integration)
- Open AI models (GPT-OSS / Llama based)
- Nosana decentralized GPU compute for inference

## 🚀 How It Works

Run `seo_ai_agent.py` to generate:
1. A list of SEO keywords for your niche
2. Content briefs for each keyword
3. A ready-to-send outreach email template
